# Coworking-site
A simple design of a website that showcases a series of co-working spaces.

![Demo1](.images/Demo1.png)

![Demo2](.images/Demo2.png)

## Design
* Tabbable navigator's icons.
* A fixed chatbox icon that always stay at the bottom right corner of the content (despite chaning the viewport's width)

## Future Implementation
* Implementing navigating links when the header's icons (Logo and Map Pin) are clicked.
* Implementing a menu interface when the hamburger icon (3 horizontal lines) is clicked.
* Implementing the chatbox interface when the chatbox icon is clicked.
* Implementing a form interface when the book icon is clicked.
